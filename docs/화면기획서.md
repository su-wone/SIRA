## 3. 대시보드 (Dashboard)

### 3.1 글로벌 통계 바

```
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│ 전체   │ │ 진행중  │ │ 리뷰   │ │ 완료   │ │ 할 일  │ │ 백로그  │
│  24    │ │   8    │ │   3    │ │  10    │ │   2    │ │   1    │
└────────┘ └────────┘ └────────┘ └────────┘ └────────┘ └────────┘
```

- 6컬럼 그리드 (반응형: sm 3컬럼, mobile 2컬럼)
- 각 카드: 라벨 + 큰 숫자 + 상태 색상

### 3.2 픽셀 오피스

```
┌─────────────────────────────────────────────┐
│ Pixel Office                                │
├─────────────────────────────────────────────┤
│                                             │
│   [수원]        [지영]         [Claude]      │
│    ☺ 💻          ☺ 💻          ☺ 💻 AI      │
│   ┌──┐          ┌──┐          ┌──┐          │
│   │  │          │  │          │  │          │
│   └──┘          └──┘          └──┘          │
│   수원           지영          Claude        │
│                                             │
└─────────────────────────────────────────────┘
```

- **Canvas 렌더링:** HTML Canvas API
- **캐릭터 구성:** 머리(8x8) + 몸(10x10) + 팔(애니메이션) + 눈
- **상태 표시:**
  - 작업중: 팔 흔들기 애니메이션 (800ms 간격) + 노란 상태 도트
  - 리뷰중: 보라 상태 도트
  - 완료: 초록 상태 도트
  - 대기: 회색 상태 도트
- **모니터:** 작업 상태에 따라 초록(작업중) / 보라(리뷰) / 어둡게(대기)
- **AI 뱃지:** 캐릭터 우측 상단에 보라색 "AI" 태그
- **태스크 툴팁:** 캐릭터 위에 현재 태스크 제목 표시 (최대 12자 + "…")
- **배경:** 격자무늬 오피스 바닥 (#111827 + #1f2937 그리드)

### 3.3 팀 현황

## 11. API 엔드포인트

| Method | Path | 기능 |
|--------|------|------|
| GET | `/api/tasks` | 전체 태스크 + 팀 목록 조회 |
| GET | `/api/tasks/[id]` | 태스크 상세 (frontmatter + 본문) |
| PATCH | `/api/tasks/[id]` | 태스크 frontmatter 업데이트 |
| GET | `/api/sse` | SSE 실시간 이벤트 스트림 |
| POST | `/api/webhook/github` | GitHub Webhook 수신 |
| GET/POST | `/api/auth/[...nextauth]` | GitHub OAuth 인증 |

### SSE 이벤트 형식

| 이벤트 | 발생 시점 |
|--------|----------|
| `task:created` | 새 태스크 파일 감지 |
| `task:updated` | 태스크 파일 수정 감지 |
| `task:deleted` | 태스크 파일 삭제 감지 |
| `index:rebuilt` | 인덱스 재빌드 완료 |

### API 응답 형식

```json
// 성공
{ "data": { ... } }

// 에러
{ "error": { "code": "NOT_FOUND", "message": "태스크를 찾을 수 없습니다" } }
```
